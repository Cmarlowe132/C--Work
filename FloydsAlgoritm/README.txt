This program will take in an input file named "input.txt" which contains the matrix for Floyd's Algorithm to run through

To run the program, compile using the command 

make Floyds

which will create the executable file Floyds which can be ran using 

./Floyds

in the console.

A file named output.txt will be created with the resulting matrix that has had Floyd's Alogorithm applied to it
